### Formatting Steps

1. Create new Google Doc
2. Name it `nav`
3. Copy-paste content above
4. Select all (Ctrl+A / Cmd+A)
5. Click bullet list icon
6. Indent sub-items with Tab:
   - Chi siamo, R&D, Sostenibilità → indent under Acerbis
   - Motorcycle, Agricoltura, etc. → indent under Settori
   - Stampaggio rotazionale, Stampaggio a iniezione → indent under Tecnologie
   - Project Management, Engineering Capability → indent under Engineering
7. Share with helix@adobe.com (Editor)
8. Publish via Sidekick

The homepage uses all Phase 1 blocks in this order:
1. Hero (with video)
2. Intro Columns
3. Technology Cards (2-column)
4. Sector Cards (5-column)
5. Engineering Cards (3-column)
6. PA6 Highlight (Columns)
7. Metadata table

## Troubleshooting

**Navigation not showing?**
- Check `nav` doc is published
- Verify helix@adobe.com has Editor access
- Reload preview

**Blocks not rendering?**
- Verify table structure (correct rows/columns)
- Check block names match folder names
- Ensure metadata table is at end of doc

**Images not optimizing?**
- Check images are in table cells
- Verify image URLs are accessible
- Try smaller file sizes (<1MB)
