# Columns Block

The Columns block creates flexible multi-column layouts with text and images, automatically responsive.

## Features

- Flexible column layout (2-3+ columns)
- Image and text side-by-side
- Responsive design (stacks on mobile)
- Image optimization
- Automatic gap adjustment

## Authoring in Google Docs

### Basic 2-Column Layout (Image + Text)

1. Insert a table with **1 row** and **2 columns**
2. **Column 1**: Image
3. **Column 2**: Heading + paragraphs + optional link

### Example: Company Description

| Image | Content |
|-------|---------|
| [Company Image] | **About Acerbis** Innovative solutions since 1973... [Learn More](link) |

### 3-Column Layout

1. Insert a table with **1 row** and **3 columns**
2. Fill each column with content

### Multiple Rows

Insert multiple rows for different column layouts on the same page.

## Content Types

### Text Column
- H2 or H3 heading
- Paragraphs
- Links (styled in blue)

### Image Column
- Single image (automatically optimized)
- Rounded corners (4px)
- Responsive sizing

## Styling

- Headings: Dark gray (#333), bold
- Paragraphs: Medium gray (#666), 1.6 line-height
- Links: Blue (#007bff), underline on hover

## Performance

- Images automatically optimized
- Lazy loading enabled
- Responsive image sizes

## Use Cases

- Company description + image
- Feature explanation + icon/image
